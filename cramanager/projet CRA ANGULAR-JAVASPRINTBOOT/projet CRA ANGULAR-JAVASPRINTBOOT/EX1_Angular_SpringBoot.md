
# EX1 — Angular + Spring Boot

Stack :
- Angular (dernier LTS)
- Spring Boot 3
- Base : H2 ou PostgreSQL

---

## 🎯 Objectif
Réaliser une application CRA full‑stack entreprise‑ready.

---

## 🗓️ Planning conseillé (4 jours)
### Jour 1 — Backend Core
- Modélisation :
  - User
  - Mission
  - Assignment
  - CRA
  - CRA Days
- Auth JWT simple
- CRUD Collaborateurs
- CRUD Missions

### Jour 2 — Backend Avancé
- Affectations missions
- Workflow CRA :
  - Création auto
  - États
- Règles 22–28
- API validation admin

### Jour 3 — Front Angular
- Auth + guards
- Dashboard Admin :
  - Liste collabs
  - Missions
  - Validations
- Dashboard Collaborateur :
  - Vue mensuelle CRA
  - Remplissage rapide

### Jour 4 — Finition
- Gestion erreurs métier
- UX propre (Material conseillé)
- README + démo
